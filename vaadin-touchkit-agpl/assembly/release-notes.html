Overview

Vaadin TouchKit adds components, utilities and a theme tailored for touch devices for use with Vaadin Framework 7.

Package Contents

Inside the ZIP installation package you will find:

  • TouchKit JARs
  • The TouchKit manual (a chapter of Book of Vaadin)
  • README and licensing information

See the README.html in the installation package for more information about the package contents. The bundled manual gives more detailed instructions.

New in Vaadin TouchKit ${Implementation-Version}

Vaadin TouchKit ${Implementation-Version} works around issues introduced in iOS 7, particularly when an app is added to the device homescreen.

  • Ticket 12795 "iOS 7 homescreen causes multiple problems"

For details, please see the relevant changeset

Known Issues

  • The initial release of iOS 7 had broken javascript Alert, Confirm and Prompt, which iOS 7.0.3 has since fixed. However, if your app uses any of these, you might want to have a look at the iOS 7 Hotfixes add-on which addresses these issues.
